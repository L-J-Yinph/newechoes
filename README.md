# 🚀 newechoes - 个人静态博客系统

## ✨ 功能特点

- 📑 **文章管理**：支持 Markdown & MDX 语法  
- 💡 **项目展示**：展示你的个人项目与作品  
- 🎬 **观影记录**：记录观影心得，分享影评  
- 📚 **读书记录**：管理阅读书单，留下笔记  
- 🌍 **旅行足迹**：记录旅行见闻，分享旅途故事  
- 🚀 **Vercel 部署**：支持国内外访问，极速体验
